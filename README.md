# 🗳️ Smart Voting System using Face Recognition

This project is a secure and efficient voting system that uses face recognition technology to prevent duplicate voting and ensure fairness in elections.

## 🚀 Features
- Face recognition using OpenCV  
- Prevents multiple voting by the same person  
- User authentication system  
- Admin panel for monitoring votes  
- Automatic vote counting  

## 🛠️ Technologies Used
- Python  
- OpenCV  
- Tkinter / Flask  
- SQLite  

## 🔮 Future Improvements
- Improve accuracy using deep learning models  
- Develop web-based interface  
- Add data encryption for better security
